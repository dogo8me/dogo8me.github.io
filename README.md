# work_tracker

A mobile-friendly dark mode web app for tracking farm work hours and notes.

## Features

- Start/stop timer for active work sessions
- Notes for each session
- Add historical time entries from past work
- Totals dashboard:
  - total hours
  - total money earned at **$10/hour**
  - average hours per work day
  - session count
- Export entries as a CSV spreadsheet
- Print-friendly report view
- Data persistence with browser local storage

## Run

Open `/home/runner/work/work_tracker/work_tracker/index.html` in your browser.
